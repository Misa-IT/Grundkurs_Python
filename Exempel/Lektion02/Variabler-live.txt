Variabler är saker vi sparar i datorns minne.
Datatyper är hur vi sparar data.


Syntax = Grammatik


Typer av datatyper:
Typer av lådor som bara innehåller en typ av sak.
str = String = Text Ex: "Hej"
int = Integer = Heltal Ex: 2
float = Float = Flyttal/Decimaltal Ex: 2.0
bool = Boolean = Sant/Falskt

Samlingsdatatyper:
lst = List = lista Kan ses på som en hylla, allt står i en viss ordning.
tuple = Kan ses på som en hylla där nån har summerlimmat fast allting.

dict = Dictionary = Ordlista
nyckel:värde t.ex. 2:"två", "ålder":32
johan={"namn":"Johan", "ålder":32, "längd":180}

set = Set En låda där det bara får finnas en sak av varje sak.


Konkatenering

Casting eller Typecasting
int(my_variable)
Ge mig värdet av my_variable som ett heltal.

str(my_variable)
Ge mig värdet av my_variable som text.
