Program 1, via pseudokod:

1. Välkomna användaren.
2. Fråga om användarens namn.
3. Om användaren svarar "hemligt", hoppa till steg 10.
4. Hälsa användaren med det angivna namnet.
5. Registera att vi har hurrat 0 gånger.
6. Hurra.
7. Registrera att vi har hurrat en gång mer än tidigare.
8. Om vi hurrat 4 gånger, hoppa till steg 11.
9. Hoppa till steg 6.
10. Hälsa användaren med en standardfras.
11. Avsluta programmet.

Program 2, Implementation i BASIC:

10 PRINT "Välkommen!"
20 INPUT "Vad heter du" namn$
30 IF namn$ = "hemligt" THEN GOTO 100
40 PRINT "Ett fyrfaldigt leve för " + namn$  + "!"
50 LET hurrarop = 0
60 PRINT "Hurra!"
70 LET hurrarop = hurrarop + 1
80 IF hurrarop = 4 THEN GOTO 110
90 GOTO 60
100 PRINT "Din integritet är viktig för oss. Ha en bra dag."
110 QUIT
